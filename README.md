## 中文数据集

中文数据集采用 [THUCNews](http://thuctc.thunlp.org/) 的一个子集进行训练， 该子集中包括了10个分类，每个分类6500条数据：

```
['房产', '科技', '财经', '游戏', '娱乐', '时尚', '时政', '家居', '教育', '体育']
```

数据集划分如下：
> - train： 5000 * 10
> - dev: 500 * 10
> - test： 1000 * 10
